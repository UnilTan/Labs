# Быстрый старт - MilkProductsWPF

## Подготовка базы данных

### Вариант 1: Использование существующей базы данных
1. Используйте существующую базу данных из проекта MilkProductsCatalog
2. Выполните скрипт `Database/TestAndCreateTables.sql` для добавления таблицы City

### Вариант 2: Создание новой базы данных
1. Выполните скрипт `Database/CreateDatabaseWithCities.sql` для создания полной базы данных с городами

## Запуск приложения

```bash
cd MilkProductsWPF
dotnet restore
dotnet build
dotnet run
```

Или запустите исполняемый файл:
```bash
.\bin\Debug\net6.0-windows\MilkProductsWPF.exe
```

## Интерфейс ProductWindow (обновлен)

### Поля ввода (6 полей в одну строку):
1. **ID продукта** - для редактирования/удаления
2. **Название продукта** - обязательное поле
3. **Цена** - обязательное поле
4. **Категория** - обязательное поле  
5. **Срок годности (дни)** - необязательное поле
6. **Город производства** - выбор из списка городов

### Основные операции

**Кнопки операций (GroupBox):**
- **Ins** - Добавить новую запись
- **Save** - Сохранить изменения
- **Del** - Удалить запись по ID
- **Sort** - Сортировать данные
- **Find** - Найти по тексту в поле поиска
- **Filter** - Применить предустановленный фильтр

**Дополнительные кнопки:**
- **Все записи** - Показать все продукты
- **По категориям** - Сортировка по категориям
- **Дорогие товары** - Фильтр товаров дороже 200 руб
- **Закрыть** - Закрыть окно

### Работа с данными:

1. **Добавление:** Заполните поля (название, цена, категория обязательны) и нажмите "Ins"
2. **Редактирование:** Выберите запись в таблице, измените поля, нажмите "Save"
3. **Удаление:** Введите ID в первое поле и нажмите "Del"
4. **Поиск:** Введите текст в поле "FIND" и нажмите "Find"

## Особенности форм

### ProductWindow:
- Управление продуктами с привязкой к городам
- 6 полей ввода в одну строку (убраны пустые поля)
- Фильтры: все записи, по категориям, дорогие товары

### CityWindow:
- Управление городами производства
- Поля: ID, Название, Регион, Страна, Население
- Фильтры: все города, по регионам, крупные города

### SaleWindow:
- Управление продажами
- Фильтры: все продажи, за сегодня, крупные продажи

### SaleDetailsWindow:
- Управление деталями продаж
- ComboBox для выбора продажи и продукта
- Фильтры: все детали, по продуктам, большие объемы
