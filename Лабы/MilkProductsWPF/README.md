# Каталог молочных продуктов - WPF версия

Это WPF приложение для управления каталогом молочных продуктов с подключением к SQL Server базе данных.

## Функциональность

### Основные формы:
1. **ProductWindow** - Управление продуктами
2. **SaleWindow** - Управление продажами  
3. **SaleDetailsWindow** - Управление деталями продаж

### Возможности каждой формы:
- **Добавление** новых записей (кнопка Ins)
- **Сохранение** изменений (кнопка Save)
- **Удаление** записей (кнопка Del)
- **Сортировка** данных (кнопка Sort)
- **Поиск** по различным полям (кнопка Find)
- **Фильтрация** данных (кнопка Filter)

### Дополнительные функции:
- Выбор записи в DataGrid автоматически заполняет поля ввода
- Специальные фильтры для каждой таблицы
- Валидация данных при вводе
- Обработка ошибок с информативными сообщениями

## Структура базы данных

### Таблица Product:
- ProductId (int, PK)
- ProductName (string, 100)
- Price (decimal)
- Category (string, 50)
- ExpiryDays (int)
- Description (string, 255)

### Таблица Sale:
- SaleId (int, PK)
- SaleDate (DateTime)
- CustomerName (string, 100)
- TotalAmount (decimal)

### Таблица SaleDetails:
- SaleDetailId (int, PK)
- SaleId (int, FK)
- ProductId (int, FK)
- Quantity (int)
- UnitPrice (decimal)

## Технологии

- **WPF** (.NET 6.0)
- **Entity Framework Core** 6.0.25
- **SQL Server** / SQL Server Express
- **XAML** для интерфейса

## Запуск проекта

1. Убедитесь, что установлен SQL Server или SQL Server Express
2. Создайте базу данных используя скрипты из папки Database
3. Обновите строку подключения в SalesContext.cs если необходимо
4. Запустите проект

## Особенности интерфейса

Интерфейс создан по образцу технического задания:
- Поля ввода в верхней части окна
- DataGrid для отображения данных
- GroupBox с кнопками операций (голубой фон)
- Поле поиска под DataGrid
- Дополнительные кнопки фильтрации внизу

Каждая форма имеет одинаковую структуру для удобства использования.
